🔧 Project Gaps & Roadmap to Funding

An honest assessment of what still needs to be built before AgriBridge is fully investor-ready.

🔴 Critical — Must Fix Before Launch

🔐 No Real Auth Persistence
JWT stored in localStorage but no "remember me", no token refresh, no logout from all devices. Users get logged out randomly.
Fix: Add refresh tokens + secure httpOnly cookies
💰 Payments in Sandbox Mode
MTN MoMo and Airtel Money are sandbox — no real money moves. Orders placed but payment never actually collected.
Fix: Register MTN sandbox → production at developers.mtn.com
📱 SMS OTP Not Sending
Africa's Talking is sandbox — OTPs are printed to server logs, not sent to phones. New users can't verify their numbers.
Fix: Sign up free at africastalking.com, add AT_API_KEY to Render
🗄️ Database Resets on Render
SQLite on Render free tier: every redeploy wipes the database. All registered users and orders are lost.
Fix: Add a free PostgreSQL database on Render (persistent storage)

🟡 Important — Needed for Real Users

📸 No Image Upload for Listings
Farmers can't upload photos of their produce. Currently uses stock images. Real photos build trust and increase orders.
Fix: Add Cloudinary free tier upload (5GB free, auto-resizes)
📋 Admin Dashboard Missing Key APIs
Admin dashboard calls /api/admin/overview, /api/admin/deliveries etc. which don't exist in app.py yet.
Fix: Add missing admin endpoints to app.py (7 routes needed)
🌐 No Custom Domain
"agribridge-1-og7a.onrender.com" looks unprofessional for investors. Need agribridge.ug or agribridge.africa.
Fix: Buy domain at UGX ~50k/yr, connect to Cloudflare Pages + Render
🔔 No Push Notifications
Farmers and buyers need WhatsApp or SMS alerts when orders are confirmed or deliveries update in real time.
Fix: Africa's Talking webhooks + WhatsApp Business API (free sandbox)
📊 Price Data Is Static
All prices shown are hardcoded in the frontend. The real /api/prices endpoint exists but isn't connected to the charts.
Fix: Pull real prices from /api/prices/summary on chart load
🚚 No Real Delivery Assignment
Delivery creation tells users to call a phone number. No in-app driver assignment, vehicle tracking, or proof of delivery.
Fix: Add driver mobile app (PWA) + Google Maps embed for tracking

🟢 Nice to Have — For Investor Demo

🗺️ Real GPS Map
Replace SVG map with real Google Maps/Leaflet showing actual farm locations
⭐ Reviews Live
Review system is built in backend but no UI to leave or view farmer reviews yet
🤝 Forward Contracts UI
The /api/contracts endpoint exists but there's no frontend form to create them
💳 Wallet Dashboard
Wallet exists in DB and API but no UI for farmers to see balance or withdraw
📱 Mobile App (PWA)
Convert to a Progressive Web App so farmers can install it on Android from browser
🔍 Real USSD Shortcode
Apply for real *789# with MTN Uganda — costs ~UGX 500K/year for dedicated code

3MTN/Airtel live payments — enables real revenue collection
4Custom domain (agribridge.ug) — professional appearance for investors
